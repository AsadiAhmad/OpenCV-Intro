# OpenCV-Intro
An Intro for OpenCV Digital Image Proccessing Library with Python Language

### Install OpenCV
```
pip install opencv-python
```
### Import Libs
```
import cv2
```
### Read Image
```
img = cv2.imread("Ahmad.jpg", cv2.IMREAD_COLOR)
```
### Show Image
```
cv2.imshow(img)
```

### Get Image Info
```
height, width, channels = img.shape
```
